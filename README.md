# 二维数组中的查找

## 题目描述

一个矩形，每一行按照从左到右递增顺序排序，每一列按照从上到下
递增顺序排序。

请完成一个函数，输入一个目标元素，返回其坐标。

## 样例

输入:

```
[[1, 2, 8, 9],
[2, 4, 9, 12],
[4, 7, 10, 13],
[6, 10, 11, 15]]
7
```

输出：

```
[2, 1]
```


## 思路

由于都是顺序排序的，所以可以根据条件来缩小搜索范围。

例如右侧的列，如果列的第一个都比搜索的数字小，那么可以直接将该列删除。

同理，可以缩小行的搜索范围。
